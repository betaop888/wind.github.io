# Винд цены (GitHub Pages)

Статический сайт с прайсом выживаемых предметов/блоков Minecraft **1.21.11**.

- Валюта: **ар**
- Базовое правило: **1 ар = 1 алмазная руда**
- Все цены: **только целые числа** (без `0.5`, `1.7` и т.д.)

## Содержимое

- `index.html` — страница сайта
- `styles.css` — тёмный минималистичный дизайн
- `app.js` — поиск, фильтры, сортировка, выдача списка
- `data/items.json` — сгенерированный прайс
- `scripts/build_items.py` — генератор цен

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub.
2. Залей содержимое папки `wind-ceny` в корень репозитория.
3. В GitHub открой `Settings -> Pages`.
4. В `Build and deployment` выбери:
   `Source: Deploy from a branch`, ветку `main`, папку `/ (root)`.
5. Сохрани. Через 1-2 минуты сайт станет доступен по ссылке GitHub Pages.

## Обновление прайса

```bash
python scripts/build_items.py
```

После генерации закоммить изменения в `data/items.json` и запушь в GitHub.
